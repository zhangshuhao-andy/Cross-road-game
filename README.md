Overview
This project includes 4 original HTML/JavaScript endless crossy-road style games designed and coded by Zhang Shuhao. All games feature skill shops, score tracking, lives, coins, dark/light mode, and smooth gameplay.
The suffixes v, s, h, d represent the 4 different game versions:
v: Desktop full-feature version (v1.4.2)
s: Mobile touch-friendly version (s1.0)
h: River-crossing special edition (h2.0)
d: 2-player neon cyber duel version (d1.7)
Game 1: Crossy Road v1.4.2 (Desktop Edition)
Full desktop game with light/dark mode
Controls: WASD / Arrow Keys / Space
Shop skills: Time Slow, Magnet, Shield, Jump Pack
Goal: Reach 100 score
Realistic shadows, vehicles, trains, and coins
Game 2: Crossy Road S1.0 (Mobile Edition)
Optimized for mobile touch
Controls: Tap forward, swipe left/right
Vibration feedback when collecting coins
Simplified shop for mobile players
Responsive fullscreen design
Game 3: Cross The River h2.0 (River Edition)
Unique water and lily pad mechanics
Lily pads sink after 1.5 seconds
New skill: Water Walk
Mix of grass, road, water, and rail terrain
Light/dark mode support
Game 4: Cross Road D1.7 (2-Player Duel Edition)
2-player competitive mode
P1: WASD | P2: Arrow Keys
Neon cyber visual style
Two players race to 100 score
Shared shop for both players
Elimination or goal-based win condition
How to Play
Open any HTML file in a browser
Click Start Game
Use keyboard (desktop) or touch (mobile) to move forward
Collect coins to buy skills
Avoid cars and trains
Reach 100 score to win
